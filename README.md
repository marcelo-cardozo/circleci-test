# CircleCI Test
Example of application built with CircleCI

### Local CircleCI
Install circleci on the computer [CircleCi](https://circleci.com/docs/2.0/local-cli/)
```shell script
make run-circleci-local
```

### Local Env
```shell script
make setup
source .udacity-devops/bin/activate
make all
```
